  • • 書籍推薦：深入淺出 jQuery
  • • 線上課程：使用 jQuery 打造互動性網頁效果

第三關：複習 HTML、CSS、jQuery

練習練習再練習，強化 HTML、CSS、jQuery 的基礎觀念。

  • • 觀念補充：完成 codecademy 的題目

    2. 2. jQuery
  • • 網頁架設：將網頁上傳到 Github Pages，讓你的網站能夠透過網址讓其他人看到
  • • 繼續練習：下載 速記表 自己找五個 jQuery Plugin 套用在網頁上

  • • 線上課程：一變應萬變的響應式網頁設計
  • • 作業練習：將 RWD 課程設計稿 製作成RWD版型
